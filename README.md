# SIGAF - Sistema Integral de Gestión Académica, Formación y Control Biométrico del SENA

Proto de alta fidelidad de un sistema ERP/CRM académico profesional diseñado para la gestión integral del SENA.

## ✅ Características
- ✅ 15-20 pantallas conectadas
- ✅ Diseño de alta fidelidad (High Fidelity)
- ✅ Paleta Verde SENA + diseño moderno
- ✅ Múltiples roles (Administrador, Instructor, Coordinador, Aprendiz, Monitor)
- ✅ Componentes profesionales (gráficos, tablas, calendarios)
- ✅ Navegación funcional
- ✅ Responsive design
- ✅ Datos ficticios realistas

## 👥 Roles del Sistema

### Administrador
- Dashboard completo con KPIs
- Gestión de usuarios
- Reportes avanzados
- Configuración del sistema
- Historial de auditoría

### Instructor
- Gestión de fichas
- Registro de calificaciones
- Seguimiento académico
- Carga de evidencias
- Generación de reportes

### Aprendiz
- Visualizar horarios
- Ver calificaciones
- Cargar evidencias
- Control de asistencia
- Consultar novedades

### Monitor/Vocero
- Reportar novedades
- Gestionar incidencias
- Comunicados del grupo
- Bitácora diaria
